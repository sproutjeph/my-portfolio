web development

"Web development" usually refers to the main non-design aspects of building Web sites: writing markup and coding. Web development may use content management systems (CMS) to make content changes easier and available with basic technical skills.

web design

Web design (page design) is the art and science of building the look, feel, and how a website and web page functions in a nutshell.
Having a clear user interface and easy to use website will ultimately lead to a better user experience for your target audience.

app design

App desing consolidates the (UI) and client experience (UX). While UI fits the general style of the application (counting the tones, text styles, and general look and feel), UX centers around the genuine usefulness and ease of use.
